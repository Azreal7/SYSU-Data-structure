## 串的存储设计

### 顺序串

用数组来存储串中的字符序列。

- 非压缩形式：

|      |      |      |      |      |
| ---- | ---- | ---- | ---- | ---- |
| c    | h    | i    | n    | e    |

- 压缩形式：

|      |      |      |
| ---- | ---- | ---- |
| c,h  | i,n  | e    |

---

<font color=red>如何表示串的长度？</font>

1. 用一个变量来表示串的实际长度，同一般线性表。
2. 在串尾存储一个终结符“\0”，表示串的结尾。

### 链接串

类似链表，但是为了节省存储空间，经常使**每个节点放多个字符**。
